
# Placement Prediction System

## Objective
Predict whether a student gets placed based on academic performance and employability factors.

## Dataset
Campus Recruitment Dataset

## Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest

## Evaluation Metrics
- Accuracy
- Confusion Matrix
- Classification Report
- ROC-AUC Score

## Skills Demonstrated
- Missing Value Handling
- Data Preprocessing
- Feature Engineering
- Classification
- Model Comparison
- Feature Importance Analysis
