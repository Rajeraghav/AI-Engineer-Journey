# Heart Disease Prediction

## Objective

Build a Machine Learning model to predict whether a patient has heart disease using clinical and medical attributes.

---

## Dataset

Heart Disease Dataset (Heart.csv)

### Features

- Age
- Sex
- ChestPain
- RestBP
- Chol
- Fbs
- RestECG
- MaxHR
- ExAng
- Oldpeak
- Slope
- Ca
- Thal

### Target

- AHD (Heart Disease)

---

## Algorithms

- Logistic Regression
- Random Forest Classifier
