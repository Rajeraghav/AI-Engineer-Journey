# Student Performance Predictor

## Objective
Predict whether a student passes or fails using machine learning.

## Dataset
Students Performance in Exams Dataset

## Algorithms Used
- Logistic Regression
- Random Forest Classifier

## Libraries
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report

## Results
Random Forest achieved higher accuracy than Logistic Regression.

## Skills Demonstrated
- Data Preprocessing
- Feature Engineering
- Label Encoding
- Classification
- Model Evaluation
- Feature Importance Analysis
